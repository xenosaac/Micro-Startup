# Backlog

Use one task per line.

Format:

`- TASK-001 | target=any-writer | priority=P1 | title=Implement onboarding CTA`

Target options:
- `any-writer`
- `role:<role-id>`
- `label:<label>`

Example tasks:

- TASK-001 | target=any-writer | priority=P1 | title=Implement onboarding CTA
- TASK-002 | target=label:ui | priority=P2 | title=Refine search results layout
